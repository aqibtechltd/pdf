How to Add Watermark to PDF:

  1. Upload your PDF file
  2. Choose watermark type:
    • Text: Enter custom text, choose font size and color
    • Image: Upload an image file and adjust size
  3. Customize watermark:
    • Adjust opacity and rotation
    • Select position on the page
    • Choose which pages to watermark
  4. Preview the result
  5. Click "Apply Watermark" to process
  6. Download your watermarked PDF
Note: All processing is done in your browser. Your files are never uploaded to any server.
