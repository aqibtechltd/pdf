How to Extract Text Using OCR:

  1. Upload your scanned PDF file
  2. Choose OCR options:
    • Select language of the text
    • Choose output format (searchable PDF or text file)
    • Specify page range (optional)
  3. Preview the document
  4. Click "Start OCR" to process
  5. Download the processed file
Note: All processing is done in your browser. Your files are never uploaded to any server.
