How to Add Page Numbers:

  1. Upload your PDF file
  2. Choose numbering options:
    • Select position on the page
    • Choose number format (1,2,3 or i,ii,iii etc.)
    • Set starting number
    • Adjust font size and color
    • Specify page range (optional)
  3. Preview the result
  4. Click "Add Page Numbers" to process
  5. Download your numbered PDF
Note: All processing is done in your browser. Your files are never uploaded to any server.
